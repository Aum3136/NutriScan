# **App Name**: NutriSnap

## Core Features:

- Image Capture: Capture food images using the device camera.
- Food Identification: Identify food items in the captured image. This feature uses a tool for determining if the identified items should trigger nutritional analysis.
- Nutritional Analysis: Display a nutritional breakdown of the detected food item including calories, protein, carbs, and fats.
- Portion Size Adjustment: Allow users to adjust portion sizes to reflect accurate nutritional information.
- Scan History: Locally store recent scan history with image thumbnails and basic food info.
- Settings Customization: Allow users to toggle dark mode and select units (grams/ounces).
- Tips Display: Provide users with helpful tips, e.g. by prompting what foods may be scanned effectively.

## Style Guidelines:

- Primary color: Fresh green (#90EE90), representing health and nature.
- Accent color: Warm orange (#FFA500) for call-to-action elements, providing energy.
- Background color: Soft off-white (#FAF9F6) for a clean and modern feel.
- Dark Mode Background: Dark Gray (#333333) for reduced eye strain in low light conditions.
- Body and headline font: 'Poppins', sans-serif for a modern, geometric, precise feel.
- Use clear and modern icons from Material Design for key actions and categories.
- Employ a clean layout using Material Design 3, with rounded cards and soft shadows for a modern UI.
- Implement smooth transitions and loading animations for an engaging UX.
